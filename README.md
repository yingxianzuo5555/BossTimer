# 🎮 Boss 刷新计时器

记录游戏 Boss 刷新时间的网页工具。免安装，打开即用。

## 功能

- ⏱ 实时倒计时，自动按刷新时间排序
- 🔴 10 分钟内即将刷新 → 红色脉冲动画警告
- 🟢 已刷新 → 绿色置顶
- ⚔ 一键击杀，自动计算下次刷新时间
- 📋 掉落信息富文本编辑（多色混排）
- 🔊 5 分钟前自动播放提示音（可自定义音频）
- 🔐 密码保护：游客只读，输入密码解锁编辑

## 使用

直接访问部署好的网址，或双击 `BossTimer.html` 本地打开。

- **默认密码**：`admin`
- 登录后在设置中可修改密码

## 部署

支持 Cloudflare Pages / GitHub Pages 一键部署。
